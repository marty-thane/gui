# React Native

Následující dokument slouží jako opora pro výuku v předmětu KI/GUI.

## Jak spustit

1. Naklonujte si repozitář:
   ```bash
   git clone https://github.com/marty-thane/gui
   ```
2. Spusťte pomocí Dockeru:
   ```bash
   docker compose up
   ```
